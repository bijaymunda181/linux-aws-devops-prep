## 1. git init :- convert a folder into working directory.
## 2. git bash :- git bash is terminal which is used to write git command.
## 3. git add "file-name" :- move untracked file into staging area.
## 4. git status :- It will show you the status of the result of command .
## 5. git add . :- Move all untracked file into staging area.
## 6. git commit -m "commit-message" :- Move all file into local repository.
